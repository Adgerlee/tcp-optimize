## 🚀 使用方法

# 2. 添加执行权限
chmod +x tcp-optimize.sh

# 3. 运行（推荐用于跨境优化）
sudo ./tcp-optimize.sh --target=global

# 或者（自动优化）
sudo ./tcp-optimize.sh --target=auto

# 或者（本地优化）
sudo ./tcp-optimize.sh --target=local
